A navigation system intended to assist visually impaired users, using a smartphone and a Microsoft Kinect. The smartphone''s magnetometer reading was used to determine which way the user was facing. Additionally, Google Maps API was used to determine the route to the destination. The Kinectic was used to determine obstacles and provide navigation at the micro-level. The smartphone provided haptic feedback informing the user whether they should move forward, turn left, or turn right.
